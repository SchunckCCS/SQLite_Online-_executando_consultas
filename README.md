# 📊 SQLite Online – Executando Consultas

Este repositório contém exercícios e práticas realizadas durante o curso **SQLite Online – Executando Consultas**.

O objetivo deste projeto é praticar comandos SQL e compreender como criar, manipular e consultar dados utilizando **SQLite em um ambiente online**.

Durante o desenvolvimento do curso, foram realizadas diversas consultas para entender melhor o funcionamento de bancos de dados relacionais e a linguagem SQL.

---

# 📚 Conteúdos praticados

Durante este projeto foram praticados os seguintes conceitos:

- Criação de tabelas (`CREATE TABLE`)
- Inserção de dados (`INSERT`)
- Consultas com `SELECT`
- Filtros utilizando `WHERE`
- Ordenação de resultados com `ORDER BY`
- Atualização de registros (`UPDATE`)
- Exclusão de dados (`DELETE`)
- Estrutura básica de bancos de dados relacionais

---

# 🛠 Tecnologias utilizadas

- SQL
- SQLite
- SQLite Online

# 📂 Estrutura do projeto

```
SQLite_Online-_executando_consultas
│
├── consultas.sql

└── README.md
```

Os arquivos `.sql` contêm as consultas desenvolvidas durante os exercícios do curso.

---

# ▶ Como executar

1. Acesse um ambiente de execução de SQLite online.
2. Copie o conteúdo de um dos arquivos `.sql` deste repositório.
3. Cole no editor SQL.
4. Execute as consultas para visualizar os resultados.
